# Alvarez Games - Frontend (Angular)

Interfaz de usuario para la comunidad de Alvarez Games, que permite a los usuarios enviar solicitudes y a los administradores gestionar los mensajes en tiempo real.

## 🛠️ Tecnologías
- **Angular 17+** (Standalone Components)
- **TypeScript**
- **CSS3 / HTML5**

## 🚀 Características
- **Formulario Dinámico:** Cambia entre modo registro y modo edición.
- **Sincronización en Tiempo Real:** Actualización automática de la tabla tras cada acción.
- **UX Mejorada:** Scroll suave automático hacia el formulario al editar y validaciones de botones.
- **Consumo de API:** Integración completa con el backend mediante `HttpClient`.
